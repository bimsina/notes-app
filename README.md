# Note Taking App made in Flutter with Sqlite database 
 This is a note taking app made with flutter.<br>
## Concepts used:
<ul>
<li>Sqlite database to store custom Note object.</li>
<li>Search Functionality</li>
<li>Staggered GridView</li>
</ul>
